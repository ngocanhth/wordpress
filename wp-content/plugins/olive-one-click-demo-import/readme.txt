== Description ==
Olive One Click Demo Import is WordPress content import which specially works best for Full Site Editing (Gutenberg block) themes. It can import global styles and settings for FSE(Gutenberg block) site.

Olive One Click Demo Import works with any WordPress themes or you can install any [Olive Themes](https://olivethemes.com/) and import [any demo](https://demo.olivethemes.com/) from the plugin.

### How to import

* Goto Appearance > Olive Importer
* On "Import" Tab select xml file in first field and select json file in second field
* Click "Start Import" button and it will automatically start importing content, settings and style.

== Installation ==
= Using The WordPress Dashboard =
* Navigate to the 'Add New' in the plugins dashboard
* Search for "Olive One Click Demo Import"
* Click Install Now
* Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =
* Navigate to the 'Add New' in the plugins dashboard
* Navigate to the 'Upload' area
* Select olive-one-click-demo-import.zip from your computer
* Click 'Install Now'
* Activate the plugin in the Plugin dashboard

= Using FTP =
* Download olive-one-click-demo-import.zip
* Extract the olive-one-click-demo-import directory to your computer
* Upload the olive-one-click-demo-import directory to the /wp-content/plugins/ directory
* Activate the plugin in the Plugin dashboard
